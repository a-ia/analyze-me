# analyze-me

A Node.js script that takes a block of text, sends it to Claude AI, and returns a structured JSON response with a summary and 3 key action items.

## Setup
1. Clone the repo
2. Run `npm install`
3. Create and add your Anthropic API key to `.env`
4. Run `node analyze.js`

## Stack
- JavaScript
- Node.js
- Anthropic Claude API
